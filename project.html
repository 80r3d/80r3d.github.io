2D & 3D Graphics Engine

The main objective of the project was to display a cube on LCD through 3D Graphics implementation on LPCXpresso 1769. The goal of this project was to learn how to interface display device with LPCXpresso 1769, to use LPCXpresso IDE and to program the CPU module to implement 3D graphics.

The LPCxpresso LPC1769 module is a development module which has ARM Cortex-M3 Microcontroller in it. A 1.8‚Äù LCD is interfaced with the LPCXpresso 1769. 18-bit (262,144) color and has 4 or 5 wire SPI digital interface. The function to display line on the LCD is called with the help of SPI.

Required strong math skills especially in vector algebra. Enlightened about various features and fuctions of LPC1769. Skilled up with programming in embedded c/c++ and learned about I/O protocols of a microcontroller and display device.

Security System Using face Detection & Recognition

This was my final year project in undergrad. In this we Implemented and deployed a face recognition security system utilizing computer vision onRaspberry Pi. Used a CSI (Camera Serial Interface) to interface pi-camera to Raspberry Pi.

I was part of 2-student team and personally worked on Raspberry pi setup and pi camera interfacing. I wrote the working python script for face detection using opencv library. I analyzed various face detection algorithms (Eigen-face, SURF, SIFT, Haar-Like) and trained machine using each model to cultivate a better idea of why selecting a particular algorithm.
